4. What is an expression?
    Expressions consist of values and operators, and they can always evaluate down to a single value.
5. What is the difference between an expression and a statement?
    Expressions evaluate to a single value, statements assign those values for later use
6. What does the variable bacon contain after the following code?
    bacon = 20
    bacon + 1
    ---bacon = 20, the value didn't change---
